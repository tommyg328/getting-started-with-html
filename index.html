Things to Know

The Web

The web is is made of many different components, first of which is us, the user. We type in commands on our computer which
get sent over the internet from our web browsers to servers using HTTP. The servers interperet the commands and respond
accordingly.

HTML

HTML is the basis for everything on the web and stands for Hyper Text Markup Language. HTML constist of:

Text Content - Everything you see
Markup - What it looks like
References - to other documents like images and videos
Links to other pages - Links are referred to as URLs. Clicking on links, moving deeper into web content is why its called
the Web.

HTML Markup

HTML Markup is what the content looks like. Markup consist of Tags and Attributes
defined within an Element.

An Element contains of an opening tag, a closing tag, and content inbetween the two tags.
If a tag has only an opening tag, but no closing tag, it is called a Void Tag.

Attributes are also part of an element, so also contained inside a tag. Attributes will point to a value.
Here is an example of an attribute:

< a href = "web address" > content link < /a>, where HREF is the attribute and the web address is the value
